Levante Ande

Aplicação proprietária do operador para automação de publicação de YouTube Shorts.

O que é

O Levante Ande (também referido como IA Short Lab) é uma aplicação de uso interno mantida pelo operador desta página. Sua única função é autenticar contra a conta do YouTube do próprio operador e enviar vídeos curtos (Shorts) produzidos a partir de cortes editados localmente.

Quem pode usar

A aplicação é restrita ao operador identificado abaixo. Não há cadastro público, não há área de clientes, não há coleta de dados de terceiros. Não é um produto comercial nem possui usuários externos.

Permissões solicitadas ao Google

  • https://www.googleapis.com/auth/youtube.upload — utilizada apenas para envio de vídeos ao canal autorizado pelo próprio operador, via o método oficial videos.insert da YouTube Data API v3.

Uso limitado das informações do Google

O uso das informações recebidas das APIs do Google adere à Google API Services User Data Policy, incluindo os requisitos de Limited Use.
