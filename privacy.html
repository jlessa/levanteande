Política de Privacidade

Aplicação: Levante Ande (também denominada IA Short Lab).

Última atualização: 21/05/2026.

1. Identificação do controlador

Operador responsável e único usuário da aplicação: jefferson2459@gmail.com. Não há entidade jurídica, equipe ou prestador terceirizado envolvido no tratamento dos dados.

2. Natureza da aplicação

O Levante Ande é um aplicativo de uso pessoal, executado em infraestrutura do próprio operador. Sua única finalidade é enviar vídeos curtos (Shorts) ao canal do YouTube autorizado pelo próprio operador, a partir de cortes produzidos e aprovados localmente. Não há base de usuários, não há conta pública, não há área de assinantes nem qualquer função compartilhada com terceiros.

3. Escopos OAuth utilizados e finalidade

  • https://www.googleapis.com/auth/youtube.upload — utilizado exclusivamente para upload de vídeos ao canal autenticado, via método oficial videos.insert da YouTube Data API v3. Não é utilizado para leitura de dados de terceiros, comentários, analytics ou qualquer outra operação além do envio de vídeos do próprio operador.

4. Informações recebidas do Google

  • Tokens OAuth 2.0 (access token e refresh token) emitidos pelo Google após consentimento explícito do operador.
  • Identificador interno da conta Google autorizada, retornado pelo fluxo OAuth, usado apenas para distinguir a sessão válida.

Nenhum dado pessoal de terceiros é recebido. Nenhum dado do Google é cruzado com qualquer outra base. Nenhum dado do Google é utilizado para perfilamento, treinamento de modelos de IA, publicidade ou qualquer finalidade não declarada nesta política.

5. Armazenamento

Os tokens OAuth são armazenados localmente no servidor controlado pelo operador, em volume restrito de filesystem, fora de qualquer repositório versionado ou serviço de terceiros. O acesso é limitado ao próprio operador.

6. Compartilhamento com terceiros

Nenhuma informação obtida das APIs do Google é compartilhada, vendida, alugada ou transferida a terceiros, sob nenhuma hipótese. Não há provedores externos, parceiros analíticos, redes de anúncios, processadores de pagamento ou integrações com plataformas de mercado envolvidos no tratamento desses dados.

7. Uso limitado (Limited Use)

O uso das informações recebidas das APIs do Google pelo Levante Ande adere integralmente à Google API Services User Data Policy, incluindo os requisitos de Limited Use. Especificamente:

  • Os dados são utilizados apenas para a funcionalidade declarada (upload de Shorts ao próprio canal autorizado).
  • Os dados não são transferidos a terceiros, exceto quando estritamente necessário para prover a funcionalidade declarada, em conformidade com Termos de Serviço aplicáveis, ou por exigência legal.
  • Os dados não são utilizados para servir anúncios, incluindo anúncios remarketing, personalizados ou baseados em interesses.
  • Humanos não leem os dados, exceto: (a) com consentimento afirmativo do próprio operador, (b) para fins de segurança (incluindo investigação de abuso), (c) para cumprir obrigações legais aplicáveis ou (d) para operações em que os dados foram agregados e anonimizados, e são usados apenas para fins internos.

8. Retenção e exclusão

Os tokens OAuth são mantidos enquanto o operador desejar continuar usando a aplicação. Podem ser revogados a qualquer momento pelo próprio operador em myaccount.google.com/permissions. A revogação invalida imediatamente o acesso da aplicação e os tokens locais deixam de funcionar.

9. Segurança

Os tokens são armazenados em filesystem com acesso restrito, fora de repositório público, fora de logs e fora de qualquer canal de transmissão externo. O tráfego com as APIs do Google é sempre via HTTPS. Não há painel administrativo exposto à internet pública.

10. Crianças

A aplicação não se destina a menores de idade e não coleta intencionalmente dados de crianças. Como não há usuários externos, esse risco não se materializa.

11. Alterações desta política

Eventuais alterações serão publicadas nesta página com nova data em Última atualização no topo. Alterações materiais (novo escopo, novo fluxo de tratamento) serão refletidas também na configuração OAuth do projeto levanteande.

Dúvidas, solicitações de exclusão e exercício de direitos: jefferson2459@gmail.com.
